# -专业批量邮件发送工具 v3.0
这是一个基于python语言写的轻量化，专业级别的批量邮箱发送工具，通过调用smtp接口进行批量邮箱发送，支持qq，网易，谷歌，阿里企业，等多个主流邮箱，也支持自定义smtp邮箱地址

一个基于 Python + Tkinter 的图形化批量邮件发送工具，支持多发件人轮发、收件人个性化变量、富文本编辑、附件、模板保存/加载、发送报告导出等功能。界面友好、操作简单，适合合法的邮件通知、营销或批量沟通场景。

**版本**：v3.0（完全稳定版）  
**作者**：幻城 官网：幻城云笔记：https://hcnote.cn/
**语言**：Python 3  
**界面**：Tkinter（跨平台，支持 Windows/macOS/Linux）

## 功能特点

- **多发件人支持**：添加多个邮箱账号，自动轮流使用，避免单个账号发送频率过高被限制。
- **收件人批量导入**：支持 TXT、CSV、Excel 文件导入，格式灵活（仅邮箱 或 邮箱,姓名）。
- **个性化发送**：主题和正文支持 `{name}` 变量，自动替换为收件人姓名（无姓名时默认“朋友”）。
- **富文本编辑器**：
  - 支持粗体、斜体、下划线、字体大小、文字颜色。
  - 可插入超链接和图片（图片会内嵌到邮件中）。
- **附件管理**：支持多文件附件，发送时自动添加。
- **模板功能**：一键保存当前主题、正文、附件为模板，下次直接加载。
- **高级设置**：
  - 可设置每封邮件间的随机延迟（默认 5~15 秒），降低被风控概率。
  - 支持填写代理列表（每行一个 http://ip:port），代码已预留位置（当前版本未完全启用代理发送）。
- **发送日志与报告**：
  - 实时日志显示发送进度。
  - 支持导出详细 CSV 报告（序号、邮箱、姓名、状态、失败原因、发送时间、发件人）。
- **重试机制**：每封邮件失败时自动重试最多 3 次。
- **支持主流邮箱**：QQ、163、Gmail、Outlook/Hotmail、Yahoo，以及自定义 SMTP。

## 屏幕预览（文字描述）

程序启动后为全屏窗口，包含以下标签页：
1. **发件人配置**：添加/删除发件人邮箱。
2. **收件人配置**：导入/清空收件人列表。
3. **邮件内容**：编辑主题和富文本正文（带工具栏）。
4. **附件与模板**：管理附件 + 保存/加载模板。
5. **高级设置**：延迟范围、代理列表。
6. **发送日志与报告**：实时日志、进度条、开始发送、导出报告。
